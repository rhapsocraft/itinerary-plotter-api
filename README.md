# Itinerary Plotter API

## Tech Stack

- Node Express
- Prisma + Kysely
- Typescript

## MVP 1

- Google SSO
  - implement user Identity tied to Google account
- Google Maps Integration
- Google Calendar API

## Guides

### Start server

```bash
pnpm start
```

### Start in dev mode

```bash
pnpm dev
```

### Generate Prisma Schema

```bash
pnpm prisma:generate
```

### Format repository

```bash
pnpm prettier
```
